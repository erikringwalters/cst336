Overview

Augmented Reality is the technology that allows for a user to see the physical world with virtual elements mixed in. The example of an augmented reality platform that I will use for this project is the Microsoft Hololens because it contains impressive software on the subjects that I will cover. (NOTE: Microsoft refers to the Hololens as a "mixed reality" device for marketing purposes, however I will still refer to it as augmented reality since it meets the criteria.)
